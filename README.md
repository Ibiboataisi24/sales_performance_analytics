#  Nigeria Retail Sales Performance Dashboard 

An end-to-end data analysis project: cleaning a messy retail sales dataset and building an interactive Power BI dashboard to track revenue, regional performance, and sales rep targets.

---

## Project Overview

A Nigerian retail company operating across five regions (Lagos, Abuja, Port Harcourt, Kano, Enugu) needed visibility into  sales performance. The raw data was a single messy Excel file with inconsistent formats, duplicates, and missing values — with no reporting capability.

This project transforms that raw data into a trusted dataset and a fully interactive dashboard for management decision-making.

---

## Business Questions Answered

- Which region generates the most revenue?
- Which product categories are underperforming?
- Are sales reps meeting their monthly targets?
- What are the peak and low revenue months?
- Which products are the top revenue drivers?

---

##  Tools Used

- Microsoft Excel — Data cleaning, validation, pivot tables
- Power BI — Dashboard development,visualization and data cleaning

---

## Data Cleaning Process

The raw dataset (200 records) contained the following issues, which were identified and fixed:

| Issue | Fix Applied |
|---|---|
| Mixed date formats | Standardized to DD/MM/YYYY |
| Duplicate Order IDs | Removed using Remove Duplicates |
| Inconsistent region names (lagos, ABUJA, PH) | Standardized to Title Case |
| Prices stored as text ("NGN 25000") | Converted to numbers |
| Blank cells in key columns | Filled using reference lookups |
| Inconsistent sales rep names | Trimmed and standardized |

##  Dashboard Features

The Power BI dashboard includes:

- KPI Cards — Total Revenue (396M) and Total Units Sold (4.2K)
- Line Chart — Monthly Revenue Trend (Jan–Dec 2023)
- Bar Chart — Total Revenue by Sales Rep
- Table with Conditional Formatting — Sales Rep performance vs Monthly Target, with Variance highlighted
- Bar Chart — Total Revenue by Region
- Donut Chart — Revenue Distribution by Product Category
- Bar Chart — Units Sold by Product Category
- Interactive Slicers — Category, Region, and Month
